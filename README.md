# Dennis-final-calculator
# Final Calculator Project

This Java calculator application was created by Dennis Franks Jr as the final course project.

## Project Summary

This application combines the work completed throughout the course project. It includes a user menu, calculator operations, memory functions, ArrayList collection storage, and exception handling.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Divide-by-zero handling
- Invalid input handling
- Single memory storage
- Integer collection storage
- Sum calculations
- Average calculations
- Difference calculations

## Concepts Used

- Java methods
- Loops
- ArrayList
- Exception handling
- Scanner input
- ArithmeticException
- InputMismatchException
